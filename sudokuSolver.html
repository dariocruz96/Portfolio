Sudoku Solver

Hi there Sudoku player! If you're looking at this project you might be trying to find a way to be the best at Sudoku?
Or just getting the max points in your smartphone app?
Well, I hope this can help you with that!

Introduction

A quick Introduction to bond with the friendly code we got there! Asking if you want help. You can either say "Yes" or "No". After that it will show you an example.

Changing entries

Now that you've accepted help it's time to put the numbers where you need them to be, for that you'll be asked to put the row, the column and the number that you want in that place (for example if you want to have a 7 in the top right block and on the top right of the square you would input '1' '9' '7').

Solution

Now it's your time to shine! Quickly copy the result to your Sudoku game and just take a screenshot to show your friends that you finished your Sudoku in 5 seconds! WOW

Reflection

This project works with a recursive algorithm, which is when the code runs throw empty spaces (0 values) it will try values that can be set there, but if in any place forward there aren't possible values, the code will run back and try different values in order to be possible to finish the Sudoku.
