Problem Solving

In the second semester of Foundation Year for Software Engineering I had the problem solving class where I had the chance to use C to do some basic code. I learned about variables, if statements, while/while...do/for loops, arrays, strings and functions.

Switch statements

The switch statement allows us to execute one block among many alternatives. The expression provided in the switch statement will be compared with each block case and if they match, the computer will read that block and end when it reads a break instruction. It works the same way as an if statement but should be easier to read and write.

For Loop

The for loop is a very used on, we can reduce a high number os coding lines if we use this loop and it's usually used when we have some kind of idea of how how many times do we need to run the loop.

Arrays

To store multiple values in one variable we use arrays, which the compiler will allocate memory after initializing the array and save that memory space for that array elements. To initialize an array we can either declare the array size doing for example ‘int array[5]’ , or we can declare its elements and the compiler will automatically know the space needed for the array. (int array [] = {1, 3, 5, 6}, the array will be size 4).
